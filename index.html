AI 학습 메모리 (0판 기억 중)

카드 구성

  • 숫자 1~15 각 4장 + 와일드 X 4장 = 총 64장
  • 2인: 각 16장 / 3인: 각 16장 / 4인: 각 16장
  • 5인: 각 12장 (4장 버림) / 6인: 각 10장 (4장 버림)
  • X는 대부분의 패에서 조커(와일드)로 사용 가능

패 종류

  • 싱글: 카드 1장. 예) 7, X (X=rank 16, 최강 싱글)
  • 페어: 같은 숫자 2장. 예) 5 5 / 5→X / X X
     → X X 페어 가능! rank 16으로 모든 페어 이김
     → ⚠️ X 포함 페어는 숫자 먼저, X 나중 순서로! (X→5 순서로 내면 무효)
  • 밤: 같은 숫자 3장 (숫자+X 조합 불가). 예) 9 9 9
     → X X X는 가능! (순수 X끼리만)
  • 밤투: 같은 숫자 4장 (숫자+X 조합 불가). 예) 3 3 3 3
     → X X X X는 가능! (순수 X끼리만)
  • 스트레이트: 연속 숫자 3장 이상, 선택한 순서대로 파싱 (X 최대 1장)
     → X가 첫 번째면 "바로 뒤 카드-1", 나머지면 "바로 앞 카드+1"
     → 예) X→4→5 : X=3 → rank5 / 3→4→X : X=5 → rank5
     → 예) 4→5→X : X=6 → rank6 / 3→X→5 : X=4 → rank5
     → 상대가 3→4→X (rank5)를 내면, 나는 4→5→X (rank6)로 받기
  • 페어스트레이트: 연속 숫자의 페어 3쌍 이상, 선택 순서대로 2장씩 파싱
     → 예) 1→1→2→2→3→X : X=3으로 마지막 쌍 완성 → rank 3 ✅
     → 1→1→2→2→X→3 : X가 쌍의 첫 번째 위치 → ❌
     → 1→1→3→3→2→X : 순서 뒤바뀜(3→2) → ❌
     → 쌍들이 순서대로 +1씩 올라가야 함

마인 / 리턴 ⚡

  • 마인: 상대가 싱글을 냈을 때, 그 카드와 같은 값을 2장 이상 가지고 있으면
     그 카드들을 선택해서 낼 수 있음 (예: 상대가 싱글 7 → 나는 7 7 선택 후 내기 / 상대가 X 싱글 → X X 마인 가능!)
     → "마인!" 효과음과 함께 가로채기 성공! 로그에 "OO: 마인 7 7" 표시
     → 밤/밤투로도 못 받음! 모두 패스하면 마인을 낸 사람이 선이 됨 (선 강탈)
  • 리턴: 마인을 당한 직후, 누군가 마인의 같은 rank의 싱글 1장을 가지고 있으면
     그 1장을 내서 "리턴!" 하며 즉시 다시 가로챌 수 있음 (예: 마인 rank 7 → 7 싱글 리턴 / 마인 X → X 싱글 리턴!)
     → 리턴이 나오면 로그가 전부 초기화되고, 리턴한 사람이 바로 선이 됨
     ��� 리턴은 진짜 방법이 없는 최후의 한 수! 로그에는 흔적이 남지 않음
  • 마인/리턴은 반드시 자기 차례일 때만 낼 수 있음 (밤/밤투와 마찬가지로 끼어들기 불가)
  • AI도 마인/리턴을 사용함. 단, 밤 재료(같은 값 3장 이상)나 스트레이트에 쓰일 카드는
     평소엔 아끼고, 위기 상황(선두를 빼앗기지 않으면 질 상황)에서만 사용함

진행 규칙

  • 랜덤으로 선이 결정, 선은 원하는 패를 자유롭게 냄
  • 다음 사람은 같은 유형의 더 높은 패를 내야 함
     → 스트레이트는 같은 길이끼리만 비교
  • 밤/밤투는 자기 차례에만 가능 (마인/리턴과 마찬가지로 끼어들기 불가)
  • 마인/리턴은 자기 차례에만 가능 (밤/밤투와 마찬가지로 끼어들기 불가)
  • 낼 수 없으면 패스. 모두 패스하면 마지막에 낸 사람이 새 선
  • 선이 바뀌면 로그 초기화, 아무 패나 자유롭게 낼 수 있음
  • 카드를 먼저 다 내는 순서대로 등수 결정
  • 제한시간 60초, 초과 시 자동 패스

승패

  • 2인: 먼저 다 내면 승리 / 3~6인: 낸 순서대로 1등, 2등...

FAQ

  • Q. X 싱글은 얼마나 강해요?
     A. rank 16으로 15보다 높아 가장 강한 싱글입니다.
  • Q. 5 X 페어 vs X X 페어, 누가 이겨요?
     A. X X 페어가 rank 16 → X X 페어 승!
  • Q. 밤에 X 못 쓴다는데 X X X는 되나요?
     A. 됩니다! X끼리만 구성된 밤/밤투는 OK. 단 5 5 X처럼 숫자+X 조합은 안 됩니다.
  • Q. 스트레이트에 X 몇 개까지?
     A. 최대 1장! X 위치에 따라 해석이 달라져요.
     첫 번째: 바로 뒤 카드-1 (X→4→5 : X=3, rank5)
     중간/끝: 바로 앞 카드+1 (3→X→5 : X=4 / 4→5→X : X=6)
  • Q. 페어스트레이트에도 X 1장?
     A. 네! 2장씩 선택 순서대로 파싱해요.
     예) 1→1→2→2→3→X : X=3으로 마지막 쌍 완성 ✅
     1→1→2→2→X→3 : X가 쌍 왼쪽 위치 → ❌ / 순서 뒤바뀌면 → ❌
  • Q. 밤 위에 밤투로 받을 수 있나요?
     A. 네! 단, 자기 차례일 때만 낼 수 있습니다.
  • Q. 마인은 언제 쓸 수 있어요?
     A. 상대가 싱글을 냈을 때, 내 차례라면 같은 값 카드 2장 이상으로 "마인!" 가로채기 가능.
     밤/밤투도 마인 판은 못 받습니다. 모두 패스하면 마인 낸 사람이 선이 돼요.
  • Q. X 싱글 마인도 가능한가요? X 리턴도?
     A. 네! 둘 다 가능합니다. 상대가 X 싱글을 냈으면 X 2장 이상으로 마인 가능하고,
     X 마인을 당했으면 X 싱글 1장으로 리턴 가능해요! (예: X 마인 당함 → X 싱글로 리턴!)
  • Q. 마인 당하면 끝인가요?
     A. 같은 rank 싱글 1장이 있다면 내 차례에 "리턴!"으로 즉시 재탈환 가능합니다!
     예) 마인 rank 7 → 7 싱글로 리턴 / 마인 X(rank 16) → X 싱글로 리턴 OK!
     리턴이 나오면 로그가 전부 사라지고 흔적도 안 남아요. 정말 막을 방법이 없습니다.
  • Q. 게임 도중 누가 나가면?
     A. 즉시 "게임 강제 중단" 처리되고 모두 새로고침됩니다.
  • Q. AI 모드는?
     A. 인원 선택에서 AI 2P~6P 선택! 인터넷 없이도 혼자 플레이 가능.
     AI 3P~6P 모드에서는 여러 AI들이 서로 경쟁하며 각각 독립적인 전략을 씁니다.
